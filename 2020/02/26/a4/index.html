在你的文章中添加短代码

为了提高文章可读性，增强人民代码素质….我编不下去了

在阅读这篇文章时，如果你是用的微信小程序那么请复制链接到浏览器打开，以便获得良好的浏览体验

复选框

用法

[noshortcode][checkbox 参数名="参数值"]内容[/checkbox][/noshortcode]

例子

[checkbox]默认复选框[/checkbox] [checkbox checked=”true”]已经完成的项目[/checkbox] [checkbox checked=”flase”]还未完成的项目[/checkbox]

[nocshortcode]
[checkbox]默认复选框[/checkbox]
[checkbox checked="true"]已经完成的项目[/checkbox]
[checkbox checked="flase"]还未完成的项目[/checkbox]
[/noshortcode]

标签

用法

[noshortcode][label 参数名="参数值"]内容[/label][/noshortcode]

例子

[noshortcode]
方形
[label]默认标签[/label]
[label color="indigo"]靛蓝标签[/label]
[label color="green"]绿色标签[/label]
圆形
[label color="indigo" shape="round"]靛蓝标签[/label]
[label color="green" shape="round"]绿色标签[/label]
[/noshortcode]

方形 [label]默认标签[/label] [label color=”indigo”]靛蓝标签[/label] [label color=”green”]绿色标签[/label] 圆形 [label color=”indigo” shape=”round”]靛蓝标签[/label] [label color=”green” shape=”round”]绿色标签[/label]

进度条

用法

[noshortcode][progressbar 参数名="参数值"]进度条标签内容[/progressbar]

例子

[noshortcode]
[progressbar progress="20"]默认进度条[/progressbar]
[progressbar progress="30" color="indigo"]靛蓝进度条[/progressbar]
[progressbar progress="40" color="green"]绿色进度条[/progressbar]
[progressbar progress="23"][/progressbar]
[progressbar]没有指明参数的进度条[/progressbar]
[progressbar progress="66.66"]小数进度条[/progressbar]
[/noshortcode]

提示

用法

[noshortcode][alert 参数名="参数值"]内容[/alert][/noshortcode]

示例

[noshortcode]
[alert]默认提示[/alert]
[alert color="indigo"]靛蓝提示[/alert]
[alert color="green"]绿色提示[/alert]
[alert title="我是标题" color="indigo"]带标题的提示[/alert]
[alert icon="flag" color="indigo"]带图标的提示[/alert]
[alert title="我是标题" icon="flag" color="indigo"]带标题和图标的提示[/alert]
[/noshortcode]

示例

[noshortcode][admonition 参数名="参数值"]内容[/admonition][/noshortcode]

[admonition title="我是标题" color="indigo"]靛蓝警告[/admonition]

折叠区块

用法

[noshortcode][collapse 参数名="参数值"]内容[/collapse][/noshortcode]

示例

[noshortcode]
[collapse title="默认折叠区块"]折叠的内容[/collapse]
[collapse title="靛蓝折叠区块" color="indigo"]折叠的内容[/collapse]

[collapse title="默认展开的折叠区块" collapsed="false"]折叠的内容[/collapse]
[/noshortcode]

时间线

用法

[noshortcode]
[timeline]
时间|标题|内容第一行|内容第二行|...|内容第 x 行
[/timeline]
[/noshortcode]

示例

[noshortcode]
[timeline]
1926-8-17|长者诞辰|+1s
2019-1-1|这是标题|我是内容|我是第二行内容
2019-12-31/23:59|左边的时间是换行的|在时间中，用 / 号表示一个换行符
[/timeline]
[/noshortcode]

[timeline] 1926-8-17|长者诞辰|+1s 2019-1-1|这是标题|我是内容|我是第二行内容 2019-12-31/23:59|左边的时间是换行的|在时间中，用 / 号表示一个换行符 [/timeline]

隐藏文本

这个功能就非常的amzing了

用法如下

[noshortcode][hidden 参数名="参数值"]内容[/hidden][/noshortcode]

示例

[noshortcode]
[hidden]一段隐藏的文本[/hidden]
[hidden type="background"]黑条隐藏文本[/hidden]
[hidden type="blur"]模糊隐藏文本[/hidden]
[hidden tip="你知道的太多了"]鼠标停留会有提示[/hidden]
[/noshortcode]

[hidden]一段隐藏的文本[/hidden] [hidden type=”background”]黑条隐藏文本[/hidden] [hidden type=”blur”]模糊隐藏文本[/hidden] [hidden tip=”你知道的太多了”]鼠标停留会有提示[/hidden]

Github Repo 信息卡

用法

[noshortcode][github 参数名="参数值"][/github][/noshortcode]

例子，以本站主题为例（由于github不稳定的原因当前未必能成功获取信息）

 [noshortcode][github author="abc2237512422" project="argon-theme"][/github][/noshortcode]

[github author=”abc2237512422” project=”argon-theme”][/github]
